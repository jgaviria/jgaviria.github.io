Installing Mission Control

Published February 24th 2016 by juan gaviria

This is a continuation on my last post creating a background task with Rocketjob

First, add the Rocket Job Mission Control gem to your Gemfile

gem 'rocketjob_mission_control'

Install gems:

bundle install

Add the following line to config/routes.rb in your Rails application:

mount RocketJobMissionControl::Engine => 'rocketjob'

Start the Rails server:

bin/rails s

Open a browser and navigate to the local Rocket Job Mission Control
